Tutorial

This tutorial provides a brief overview of the concepts from the backend Flask framework and the frontend Bulma CSS framework which were both used to build the Demo page. Links to the corresponding documentation are also listed to give a more in-depth explanation.

Building Routes with Flask

After following the instructions from the Install page about Flask, we can now create routes to different pages in our application. To begin doing this, make a file called app.py in your project folder and insert the following code:

from flask import Flask
app = Flask(__name__)

@app.route("/")
def index():
    return "<h1>Hello World!</h1>"

After creating this file, run the following commands in a Terminal window like Git Bash.

$ export FLASK_APP=app
$ export FLASK_ENV=development
$ flask run

After running the last command, Flask should have displayed a message showing a localhost address (127.0.0.1) and the port number to see your page in the browser. Open Chrome and enter this address to see your "Hello World!" page.

Flask Templates

To render bigger chunks of HTML, we should create templates for each page on the website. To render a template in Flask, follow these steps:

  1. Create a templates directory in the same folder as your app.py file.
  2. Add a file called base.html to the templates directory.
  3. Modify app.py to add Flask's render_template() function and add this HTML to the base.html file.
# app.py
from flask import Flask, render_template
app = Flask(__name__)

@app.route("/")
def index():
    return render_template("base.html")
<!-- base.html -->
<!DOCTYPE html>
<html>
    <head>
        <title>My Website</title>
    </head>
    <body>
        <h1>Hello World</h1>
    </body>
</html>

Since HTML content like navigation bars, headers, and footers can be repeated across many pages in our website, we can use Flask's template inheritance in order to make changes easier to maintain. This allows templates to inherit all previous HTML content from other pages.

Using Bulma's Class Styles

After attaching Bulma's CDN to our template files (mentioned in Install page), we can use Bulma's provided documentation about CSS classes to style HTML elements like headings <h1> to <h6>, images, sections, buttons, links, and others. Here are a few examples of how to style text, images, and apply colour styles.

Styling Text

Bulma provides many typography classes which can be used to change the font properties and alignment of textual elements such as headings, paragraphs, and others. For example to create a bold and centered <h1>, we can use this code inside our base.html file:

<h1 class="is-size-1 has-text-centered has-text-weight-bold">Hello World</h1>

Styling Images

Bulma's documentation about images shows ways to display images responsively with fixed sizes. For example, to create a rounded 128x128 image, we can use the following code inside our base.html file.

<figure class="image is-128x128">
    <img class="is-rounded" src="my-image.jpg">
</figure>

Adding Colour

To draw attention to certain elements or make them more attractive, we can use Bulma's colour classes to add text colour and background colour. For example to style the <h2> headings on this page, the following code was used.

<h2 class="has-text-primary-dark">Secondary Heading</h2>
