Tutorial

Building Routes with Flask

After following the instructions from the Install page about Flask, we can now create routes to different pages in our application. To begin doing this, make a file called app.py and insert the following code:

from flask import Flask
app = Flask(__name__)

@app.route("/")
def index():
    return "<h1>Hello World!</h1>"

After creating this file, run the following commands in a Terminal window like Git Bash.

$ export FLASK_APP=app
$ export FLASK_ENV=development
$ flask run

After running the last command, Flask should have displayed a message showing a localhost address (127.0.0.1) and the port number to see your page in the browser. Open Chrome and enter this address to see your "Hello World!" page.

Flask Templates

To render bigger chunks of HTML, we should create templates for each page on the website. To render a template in Flask, follow these steps:

  1. Create a templates directory in the same folder as your app.py file.
  2. Add a file called base.html to the templates directory.
  3. Modify app.py to add Flask's render_template() function and add this HTML to the base.html file.
# app.py
from flask import Flask, render_template
app = Flask(__name__)

@app.route("/")
def index():
    return render_template("base.html")
<!-- base.html -->
<!DOCTYPE html>
<html>
    <head>
        <title>My Website</title>
    </head>
    <body>
        <h1>Hello World</h1>
    </body>
</html>
