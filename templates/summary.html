Summary of Frameworks

Flask

Flask is popular among Python developers as it was voted the most popular web framework in the 2018 Python Developers Survey. Flask is often compared against another commonly-used web full-stack framework called Django. In October 2020, Flask was the runner-up to Django for the most stars on GitHub for a web-development framework written in Python.

Used by these major companies

Strengths

  • Lightweight and easy for beginners
  • Functionality can be easily added if requirements change
  • Large community and accessible documentation

Weaknesses

  • Poor performance for high website traffic (Flask is single-threaded)
  • Administration features are not as prominent
  • More complicated maintenance for complex websites

Bulma

Bulma is a popular CSS framework based on Flexbox that is often compared to other major frontend frameworks like Bootstrap and Foundation. Bulma also has over 40,000 stars on it's GitHub repository which shows it's well-received by many software developers. This framework is known for it's minimalist design and has almost 150,000 downloads every week.

Used by these major companies

Strengths

  • Developer-friendly since class names are descriptive and easy to understand
  • Easily customizable by changing values of variables
  • No JavaScript required so can be integrated with popular JavaScript frameworks

Weaknesses

  • Appearance of websites is very similar without customization
  • Less complete in terms of accessibility features compared to other CSS frameworks
  • No JavaScript means developers must manually write code to produce interactive behaviour
