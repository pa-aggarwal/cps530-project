Install

This page shows the requirements and steps of how to install Flask and get started with Bulma.

Flask

  1. You must download Python on your computer to run Flask.
    • If you have Python, it must be version 3.6+ for Flask to work correctly
    • Otherwise, get the newest release version for your specific OS
    • To check your version of Python, open a Terminal or Command Prompt and run python --version
  2. Follow the steps provided in Flask's installation page.

Bulma

  1. Open Bulma's Getting Started page and scroll down to the CDN option.
  2. Copy and paste the HTML <link> into the <head> section of any HTML file to use Bulma's style classes.
